# wechat_shop
微信小程序模版同城微商三级分销系统商城秒杀拼团裂变支付小程序
这是一款能跑起来的开源商城系统！稳定且小巧！
微信小程序电商平台（前后端开源PHP），包含拼团，分销，抽奖，红包，多店，会员制，种草社交，新零售，社区团购等功能，整个系统架构非常简单，适合小型团队或者个人开发者二次开发。

小程序 + APP + 公众号 + PC + H5 + 支付宝小程序 + 百度小程序 + 头条号小程序

注重界面美感与用户体验，打造独特电商系统生态圈
